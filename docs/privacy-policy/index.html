Privacy Policy

This extension processes Module Federation runtime data locally in your browser.

Data collected

  • No data is sent to any server.
  • No analytics, tracking, or telemetry are used.
  • No personally identifiable information is intentionally collected or stored.

What the extension accesses

The extension reads the currently active tab only when you open it and request a scan. It inspects page runtime data and loaded resource URLs to detect Module Federation remotes and shared dependency versions.

Data sharing

No scan results are transmitted off-device.

Storage

The extension does not persist scan results beyond the current popup session.

Changes

If future versions add syncing, telemetry, or remote services, this policy will be updated before release.
